pwd - Проверка текущей директории.

ls (list) — программа, которая выводит список файлов и директорий в рабочей (текущей) директории.

cd (change directory), с помощью неё осуществляется перемещение по файловой структуре.

man - смотреть документацию соответствующей программы //https://explainshell.com//

stat - Информация о любом файле или директории.

cat - прочитать файл.

head и tail - посмотреть только начало файла или его конец. // head -n 10 первые десять строк файла.

less - Особая категория программ для просмотра содержимого файла — пейджеры.

grep (global regular expression print) - выполняющей поиск по файлу или файлам определённого текста.

vim .bashrc - текстовый редактор.

ПОТОКИ:

STDIN (Standard Input) - отвечает за вывод на экран.

STDOUT (Standard Output) - вывод на экран.// > и >> 

STDIN - стандартный ввод // <

STDERR - вывод ошибок. // 

ПРИМЕР ПАЙПЛАЙНА: 

cat source | grep Dog | uniq | sort
1. Читается файл source
2. Входные данные грепаются по подстроке "Dog"
3. Убираются дубли (в исходном файле две одинаковых строки "Dog")
4. Входные данные сортируются и выводятся на экран.

touch - команда для создания файла.

rm (remove files) - удалить файл
rm -r - удаление директории.
rm -rf - удаление директории без вопросов.

mv (move) - Переименование всегда равносильно перемещению, которое выполняется командой.

cp (copy) - Для копирования файлов и директорий. //Её первый аргумент — имя источника (откуда), второй — имя приёмника (куда).

mkdir - Создание директории. // -p создание вложенных директорий.

env - Посмотреть установленные переменные. //https://ru.hexlet.io/courses/cli-basics/lessons/environment-variables/theory_unit

history - Посмотреть историю команд. // history 5, то отобразятся только 5 последних введённых команд
При необходимости историю всегда можно погрепать:
history | grep export
